# GoShopy-Server



# Prerequisite
Must have basic knowledge of  **Node**,  **React**,  **Express**,  **MongoDB**  .  

# Install Dependencies

[](https://github.com/SR22kl/GoShopy-Server?tab=readme-ov-file#install-dependencies)

**For Backend**  -  `npm i && npm run build`

## Env Variables

[](https://github.com/meabhisingh/mern-ecommerce-server-2024?tab=readme-ov-file#env-variables)

Make Sure to Create a .env file in root directory and add appropriate variables in order to use the app.

**Essential Variables**  
PORT=  `4000 or any`  
MONGO_URI=  `mongodb://localhost:27017 or cloud uri`  
STRIPE_KEY=`stripe secret key`  
PRODUCT_PER_PAGE=`8 or any`

_fill each filed with your info respectively.
